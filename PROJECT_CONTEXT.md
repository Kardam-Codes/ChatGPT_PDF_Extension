# Project Context — Hackathon (Frozen)

Project Name:
ChatGPT → Perfect PDF

Problem Statement:
Users struggle to convert ChatGPT responses into clean, professional PDFs
without losing formatting, code blocks, or readability.

Core Goal (MVP):
- Paste ChatGPT content
- Edit content
- Preview formatted output
- Export a clean PDF

Tech Stack:
- Frontend: HTML, CSS, Vanilla JS
- Backend: Node.js + Puppeteer
- Markdown parsing: marked
- Syntax highlighting: highlight.js
- No database
- No authentication

Key Features:
- Dark / Light theme
- Font size control
- Code blocks with copy button
- PDF export via local server

New Features in Progress:
- PDF color theme picker
- Font picker
- Improved export controls
- Toast notifications

Constraints:
- Single-day hackathon
- Stability > features
- Must run locally
- No external services

UI Rules:
- Minimal
- Clean
- No heavy animations
- Clear buttons and icons

Out of Scope:
- Login / accounts
- Cloud storage
- ChatGPT API integration
- Mobile app

Golden Rule:
Do NOT redesign architecture.
Do NOT refactor heavily.
Do NOT add features not listed above.
