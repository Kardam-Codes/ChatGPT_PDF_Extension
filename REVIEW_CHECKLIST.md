# Integration Review Checklist

Before merging any branch, check:

## Stability
- App loads without errors
- No console crashes
- Export still works

## Scope
- Only assigned features added
- No surprise refactors
- No removed functionality

## Code Quality
- Clear variable names
- No commented-out junk
- No duplicate logic

## Safety
- Defaults handled
- Edge cases considered
- No hard crashes

## Communication
- Commit messages are clear
- Changes explained by author
